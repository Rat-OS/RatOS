Rat Rig V-Core Pro 1.3

The Rat Rig V-Core Pro 1.3 is a (discontinued) Core XY printer in 3 large sizes produced and sold as kits by Rat Rig it has the same kinematic bed system as the V-Core 3, but based on a 2020/2040 frame.

Supported Boards​

  • BIGTREETECH Octopus v1.1
  • BIGTREETECH Octopus Pro 446
  • BIGTREETECH Octopus Pro 429
  • BIGTREETECH SKR Pro v1.2
  • Fysetc Spider v1.1

Post installation steps​

Replace your printer.cfg with config/templates/v-core-pro-printer.template.cfg
